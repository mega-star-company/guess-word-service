═══════════════════════════════════════════════════════════════
            ⚠️  IMPORTANT: How to Run the Server
═══════════════════════════════════════════════════════════════

You're seeing "ModuleNotFoundError: No module named 'fastapi'"
because you need to ACTIVATE THE VIRTUAL ENVIRONMENT first!

═══════════════════════════════════════════════════════════════
              ✅ CORRECT WAY (2 Steps)
═══════════════════════════════════════════════════════════════

STEP 1: Activate the virtual environment
─────────────────────────────────────────
cd /Users/talben-avi/guess-word/service-repo
source venv/bin/activate

→ You'll see (venv) appear in your prompt

STEP 2: Run the server
─────────────────────────────────────────
python3 main.py

═══════════════════════════════════════════════════════════════
          📝 BEFORE FIRST RUN: Get API Key
═══════════════════════════════════════════════════════════════

The server needs a Hugging Face API key (FREE!).

Quick setup:
    ./configure_api.sh

Or manually:
    1. Sign up: https://huggingface.co/join
    2. Get token: https://huggingface.co/settings/tokens
    3. Create .env file:
       echo "HUGGINGFACE_API_KEY=hf_your_token" > .env

═══════════════════════════════════════════════════════════════
           🚀 ONE-COMMAND RUN (Easiest)
═══════════════════════════════════════════════════════════════

Use the helper script that does everything:

    cd /Users/talben-avi/guess-word/service-repo
    ./run.sh

This automatically activates venv and starts the server!

═══════════════════════════════════════════════════════════════
              ❌ WRONG WAY (What You Did)
═══════════════════════════════════════════════════════════════

python3 main.py  ← WITHOUT activating venv first

This fails because Python doesn't know about the installed packages!

═══════════════════════════════════════════════════════════════
                   📚 Quick Reference
═══════════════════════════════════════════════════════════════

Always remember these 2 steps:
    1. source venv/bin/activate
    2. python3 main.py

Or just use: ./run.sh
