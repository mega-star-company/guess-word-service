═══════════════════════════════════════════════════════════════
              📊 HOW IT WORKS
═══════════════════════════════════════════════════════════════

Each day:
1. Takes current date (YYYY-MM-DD)
2. Creates MD5 hash of the date
3. Uses hash to pick one of 300 words
4. Same calculation = same word for everyone today
5. Tomorrow = different hash = different word

Result: Fresh word every day for ~300 days! 🎯
